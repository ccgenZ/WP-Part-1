# Sistem Peminjaman Fasilitas Kampus - Tanri Abeng University

Website ini adalah portal sederhana untuk memfasilitasi peminjaman ruangan dan fasilitas kampus di Tanri Abeng University. Proyek ini dibuat menggunakan **Mobirise Website Builder** sebagai bagian dari tugas kuliah.

## Deskripsi Proyek

Website ini bertujuan untuk memudahkan mahasiswa dan staf dalam melihat ketersediaan serta melakukan pemesanan fasilitas kampus. Pengguna dapat melihat daftar fasilitas umum, studio jurusan, hingga ruang kelas beserta kapasitasnya.

### Fitur Utama:
* **Beranda (Home):** Menampilkan fasilitas populer (Tax Center, Kolam Renang, Gym, dll) dan fasilitas penunjang (AC, Wifi, Proyektor), Lokasi, dan Kontak.
* **Daftar Fasilitas:** Halaman khusus yang mengkategorikan ruangan menjadi:
    * *Fasilitas Umum* (Rans Room, Perpustakaan, Musollah, dll).
    * *Studio Jurusan* (Lab Arsitektur, Lab Komputer, Studio Perminyakan).
    * *Ruang Kelas* (Dikategorikan berdasarkan kapasitas 20 dan 50 orang).
* **Galeri Kegiatan:** Dokumentasi visual penggunaan fasilitas kampus.
* **Integrasi Kontak:** Tombol langsung ke WhatsApp dan Google Maps untuk lokasi kampus.
* **Formulir Pendaftaran:** Terintegrasi dengan Google Forms untuk proses booking.

##  Teknologi yang Digunakan
* **Builder:** Mobirise v6.1.4
* **Framework:** Bootstrap 5
* **Bahasa:** HTML, CSS, JavaScript

##  Struktur File
* `index.html` - Halaman utama (Landing Page).
* `page1.html` - Halaman daftar lengkap fasilitas.
* `page4.html` - Halaman galeri foto.
* `page3.html` - Halaman riwayat (Coming Soon).
